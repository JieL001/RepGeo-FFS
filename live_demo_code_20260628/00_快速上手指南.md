# FFS / RepGeo-FFS 现场演示快速上手指南

本代码包用于现场展示项目代码，不用于替代完整工程。请将本目录放在 `RepGeo-FFS` 工程根目录下使用。

## 1. 先确认运行环境

打开 PowerShell，进入工程根目录：

```powershell
cd RepGeo-FFS
conda activate ffs
```

检查权重、数据、脚本和 CUDA：

```powershell
powershell -ExecutionPolicy Bypass -File live_demo_code_20260628\commands\00_check_demo_assets.ps1
```

预期看到：

```text
[OK] weights\23-36-37\model_best_bp2_serialize.pth
[OK] scripts\train_kitti.py
[OK] core\lora.py
cuda_available= True
```

如果 `cuda_available=False`，说明没有进入正确的 `ffs` 环境。

## 4. 三句话讲清代码贡献

1. 原 FFS 主干负责完成双目输入到视差输出，主干结构没有被重写。
2. FFSOmega-R 在训练阶段加入参数高效残差和可靠性回放，用来提升 KITTI 目标域并缓解跨域遗忘。
3. RepGeo-FFS 把训练得到的残差折叠回原始卷积权重，导出静态 FFS-style checkpoint，部署时不保留额外分支。
